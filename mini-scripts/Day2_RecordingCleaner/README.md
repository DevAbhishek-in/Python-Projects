MP3 File Manager

Description

This project is a simple Python program that helps manage MP3 files in a selected directory.

I created this program because my phone automatically records calls. After some time, there were 50–100 recording files, and deleting them manually became time-consuming.

This program scans the selected directory, creates a Recycle_bin folder, moves all MP3 files into it, and then asks whether the Recycle Bin should be permanently deleted.

Features

- Scan a selected directory
- Detect all MP3 files
- Create a Recycle_bin folder automatically
- Move MP3 files safely
- Option to permanently delete the Recycle Bin
- Simple command-line interface

Technologies Used

- Python
- os
- shutil
- time
