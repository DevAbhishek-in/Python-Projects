# 🎮 Perfect Guess Game

A simple number guessing game built with Python using Object-Oriented Programming (OOP).

The computer randomly selects a secret number between 1 and 100, and the player tries to guess the correct number with the help of hints.

## 🚀 Features

- 🎲 Random number generation
- ⬇️ Too Low hint system
- ⬆️ Too High hint system
- 🔢 Attempt counter
- ⏱️ Game time tracking
- ⚠️ Input validation
- 🛡️ Exception handling:
  - ValueError
  - KeyboardInterrupt
  - EOFError
  - Unexpected errors

## 🧠 Concepts Used

- Python Classes and Objects
- Inheritance
- Functions
- Loops
- Conditional Statements
- Exception Handling
- random module
- time module
